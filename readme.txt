This is Joseph Moreau Extra Credit blackjack script

**Required**: pip install pyfiglet for custom banner. 

How to play:
1. Launch script, would you like to play BlackJack y for Yes, and n for No
2. Cards will be dished out to you and the computer 1 for a another card, 2 to Pass
3. A typo will freeze the game, and ask you to reinput a 1 or a 2 to continue
4. Once the user or computer hits 21 or goes over the game ends, you will be asked if you if you'd like to replay. 

5. Enjoy
